Project is done by

- 23AIML056 - Kavya Prajapati

The project uses LSTM model and RAVDESS Dataset

- RAVDESS Dataset - https://www.kaggle.com/datasets/uwrfkaggler/ravdess-emotional-speech-audio
- Keras LSTM - https://keras.io/api/layers/recurrent_layers/lstm/

The Project uses GUI Based Python Script for recording audio

The Jupyter notebook contains data visualization, feature extraction to be used in the model, Keras LSTM model, Live predictions

To directly use the jupyter notebook upload it to Google Colab, then change the data paths
